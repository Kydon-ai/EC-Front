## 滚动开发目标

-[√] 1) 首屏优化（不要再控制台开启禁用缓存，后续再进行进一步优化） <br>

-[√] 2) 后端技术选型——NodeJS + MongoDB <br>

-[√] 3) RAG 的基本方案——RAGFlow <br>

-[√] 4) 前端样式模块化选型——TailWindCSS 样式组 <br>

-[√] 5) Pretty + ESLint(暂时先使用 ESLint + IDE 根据 ESLint 的方案进行格式化) <br>
